Mopt_ntuple // makefile to gmake geant batch  (obslolete, not recommended)
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
          // setenv AMSPROFILE 1  adds gprof
Mopt      // same + ability to write root files   (Recommended)
Mgxint    // makefile to gmake geant interactive  g3
Moptg4    // makefile to gmake g4 batch
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
          // setenv O4 1   super-opt option (alpha only)
          // setenv AMSWD to some tmp dir is  recommended
Moptg4i    // makefile to gmake geant interactive  g4


To create standalone simulation/reconstruction program (without db) use:

 batch                   gmake -if Mopt (Moptg4)
 interactive(g3 )   gmake -if  Mgxint  (undefine AMSICC; gmake -f Mopt first)
 interactive(g4 )   gmake -if  Moptg4i  

To use gnu g77 compiler  (not recommended,-40% program speed, still needed for interactive geant3)
unsetenv AMSICC 
gmake -if Mopt

To create production server:

 gmake server -f Mopt2   ! NB Orbit2/RH9 only


To create gbatch_producer to server:

 gmake producer -f Mopt2  ! NB Orbit2/RH9 only





To build ams "ROOT" shared library  ntuple.so

 gmake -f Mopt shared
