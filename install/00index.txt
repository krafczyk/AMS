Mopt      // makefile to gmake geant batch 
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
Mgxint    // makefile to gmake geant interactive  
Moptg4    // makefile to gmake g4 batch
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
          // setenv AMSWD to some tmp dir is  recommended


To create standalone simulation/reconstruction program (without db) use:

 batch                   gmake -if Mopt (Moptg4)
 interactive(g3 only )   gmake -if  Mgxint  (batch program should be created first)
