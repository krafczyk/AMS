Overall:

The choice of the compiler suite (gcc/g77  or icc/ifort) is automaticaly done
as a consquence of the root-config --arch command.
It then depends on the compiler type has been used for  your ROOT version. 

The new Perugia Tracker Software can be enabled at compile time with:
setenv PGTRACK 1
before compiling do a make clean.

WARNING!!: Perugia Tracker Software is under developpement, is tested only on gcc
         and it is not yet full featured.

gmake gbatch(default) // builds g3 gbatch with root
gmake gxint           // builds g3 gxint !undefine AMSICC first
gmake lib             // builds root shared & static library
gmake shared          // builds root shared library
gmake static          // builds root static library
gmake producer        // builds gbatch-orbit (add -i flag for Intel Compilers mode)
gmake server          // builds server       (add -i flag for Intel Compilers mode)

gmake -f Moptg4   //builds g4 batch (not supported at the moment)
