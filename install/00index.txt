Mopt      // makefile to gmake geant batch 
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
Mopt_Root // same + ability to write root files
Mgxint    // makefile to gmake geant interactive  g3
Moptg4    // makefile to gmake g4 batch
          // setenv AMSDEBUG 1 debug version; unsetenv AMSDEBUG opt version
          // setenv O4 1   super-opt option (alpha only)
          // setenv AMSWD to some tmp dir is  recommended
Moptg4i    // makefile to gmake geant interactive  g4


To create standalone simulation/reconstruction program (without db) use:

 batch                   gmake -if Mopt (Moptg4)
 interactive(g3 )   gmake -if  Mgxint  (batch program should be created first)
 interactive(g4 )   gmake -if  Moptg4i  (batch program should be created first)


To create production server:

 gmake server -f Mopt


To create gbatch_producer to server:

 gmake producer -f Mopt
